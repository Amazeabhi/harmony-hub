## üßê Overview

This project was built to explore the complexities of handling **audio state management**, **file storage**, and **relational databases**. It provides a premium-feel interface with smooth transitions and real-time data fetching.

### üåü Key Features

* **Custom Audio Player:** Advanced player controls (Play/Pause, Skip, Volume, Progress Bar).
* **Authentication:** Secure Login/Signup via Google, GitHub, or Email.
* **Song Management:** Users can upload their own tracks and cover art.
* **Library System:** "Like" songs to add them to your collection instantly.
* **Smart Search:** Filter songs by title, artist, or genre.
* **Glassmorphism UI:** A sleek, modern interface built with Tailwind CSS.
